(1)色の指定

Top Before Next Last

色の指定は英単語(キーワード)、もしくは数値で行う。

■英単語(キーワード)での指定

  • color:red;

  • color: blue;

■数値での指定

  • RGB指定。各色の1桁目と2桁目が同じであれば省略可

    color:#f00;

  • RRGGBB指定

    color:#ff0000;

  • RGB指定

    color:rgb(255,0,0);

  • RGBA指定。Aは透明度(指定可能範囲0～1)

    color:rgba(255,0,0,0.5);

  • 完全に透明にする場合、transparentを指定する(color:transparent;)

    color:transparent;

  • HSL指定。

    HSLは「色相」、「彩度」、「明度」で指定する。 色相の単位は度で範囲は0～360度。赤は0度、緑は120度、 青は240度と定められている(CSS上は度の単位は記載しない)。

    彩度は色の鮮やかさを表し、低いとくすんだ色になる。 彩度の単位は%で範囲は0～100%。

    明度は色の明るさを表し、高くなるほど明るくなる。 明度の単位は%で範囲は0～100%。

    color:hsl(0, 100%, 50%)

  • HSLA指定。HSLに透明度を加えたもの。透明度の範囲はRGBAと同様0～1。

    color:hsla(0, 100%, 50%, 0.5)

  • opacityプロパティに透明度を指定可能。範囲は0～1。

    opacity:0.5;

(2)単位の指定

Top Before Next Last

■相対単位

基準となる値に対して相対的に大きさが決まる。 基準となる値が変化すると、相対単位で指定されて値も変化する。

  • %

    その要素の親要素のサイズを100%として算出される。

    style="width:400px;"
    style="width: 50%;" = 200px
  • em

    その要素の親要素に対する割合で算出される。

    style="font-size:1.5em;"

    style="font-size:1em;"

    style="font-size:0.5em;"

  • rem

    HTMLの単位に対する割合で算出される。

    style="font-size:1.5rem;"

    style="font-size:1rem;"

    style="font-size:0.5rem;"

■絶対単位

  • px

    スクリーンディスプレイではディスプレイの1ドット、 高密度ディスプレイなど1ピクセルが倍の密度になっている場合、 物理的なピクセル数の横幅が640pxだとしても、 CSSでは320pxとして算出するルールとなっている。

    style="font-size:15px;"

(4)擬似クラス

Top Before Next Last

特定の外的要因や状態を持つ要素に対してスタイルを適用する。

  • :link

    全てのリンクにスタイルを適用

    a:link{font-weight: bold;}

  • :visited

    訪問済リンクにスタイルを適用。 プライバシー上の理由で適用できるスタイルはブラウザが厳しい制限をかけている。 color、background-color、border-color、border-bottom-color、border-left-color、 border-right-color、border-top-color、outline-color、column-rule-color、 fill、strokeのみ適用され、アルファ値は無視される

    a:visited{font-weight: bold;}

  • :active

    アクティブなリンクにスタイルを適用

    a:active{font-weight: bold;}

  • :hover

    マウスカーソルが要素の上に乗ったときにスタイルを適用。 タッチスクリーンでは :hover には問題があるか、使えない

    a:hover{font-weight: bold;}

  • :focus

    フォーカスが要素にあたったときにスタイルを適用

    a:focus{font-weight: bold;}

  • :first-child

    指定要素の最初の子要素にスタイルを適用

    .classname:first-child{font-weight: bold;}

  • :last-child

    指定要素の最後の子要素にスタイルを適用

    .classname:last-child{font-weight: bold;}

  • :target

    ページ内リンクのターゲット対象にされた要素にスタイルを適用 (ページ内リンクがクリックされた際にスタイルが適用される)

    p:target{font-weight: bold;}

  • :enabled

    指定要素が使用可能な状態のときにスタイルを適用

    button:enabled{font-weight: bold;}

  • :disabled

    指定要素が使用不可能な状態のときにスタイルを適用

    button:disabled{font-weight: bold;}

  • :checked

    ラジオボタン、チェックボックスがチェック状態のときにスタイルを適用

    input:checked{font-weight: bold;}

  • :nth-child(n)

    指定要素の最初からn番目の要素にスタイルを適用。 指定要素以外の要素もカウントされるため、 指定要素以外の要素がn番目にあたる場合、 スタイルは適用されない

    li:nth-child(2){color: red;}

  • :nth-last-child(n)

    指定要素の最後からn番目の要素にスタイルを適用。 指定要素以外の要素もカウントされるため、 指定要素以外の要素がn番目にあたる場合、 スタイルは適用されない

    li:nth-last-child(2){color: red;}

  • :nth-of-type(n)

    指定要素の最初からn番目の要素にスタイルを適用。 指定要素以外の要素はカウントされない

    li:nth-of-type(2){color: red;}

  • :nth-last-of-type(n)

    指定要素の最後からn番目の要素にスタイルを適用。 指定要素以外の要素はカウントされない

    li:nth-last-of-type(2){color: red;}

  • :first-of-type

    指定要素の最初の要素にスタイルを適用。 指定要素以外の要素はカウントされない

    li:first-of-type{color: red;}

  • :last-of-type

    指定要素の最後の要素にスタイルを適用。 指定要素以外の要素はカウントされない

    li:last-of-type{color: red;}

  • :only-child

    親要素に他の要素が含まれていない場合に指定要素にスタイルを適用

    div p:only-child{color: red;}

  • :only-of-type

    親要素に指定要素が複数含まれていない場合に指定要素にスタイルを適用

    div p:only-of-type{color: red;}

  • :empty

    子要素がない指定要素にスタイルを適用

    td:empty{color: red;}

  • :not(s)

    sで指定した要素以外にスタイルを適用

    div:not(.test){color: red;}

※カウントの仕方について 

<div class="test">
  <p>1番目</p>
  <p>2番目</p>
  <span>3番目</span>
  <p>4番目</p>
  <p>5番目</p>
</div>

div.test p:nth-child(3)、 div.test p:nth-last-child(3)の場合、 span要素が該当要素となるため、スタイルが適用されない。

div.test p:nth-of-type(3)の場合、4番目にスタイルが適用される。 div.test p:nth-last-of-type(3)の場合、2番目にスタイルが適用される。

このカウントの考え方は以下のセレクタに共通する。

  • :nth-child(n)と:nth-of-type(n)
  • :nth-last-child(n)と:nth-last-of-type(n)
  • :first-childと:first-of-type
  • :last-childと:last-of-type
  • :only-childと:only-of-type

(5)擬似要素

Top Before Next Last

特定のドキュメントにスタイルを適用する。 CSS3から:ではなく、::を使用するように変更された。 (:も使用可能だが、::selectionには使用できない)

  • ::before

    指定要素の前にcontentで指定した内容を付加

    p::before{content: "★";}

  • ::after

    指定要素の後ろにcontentで指定した内容を付加

    p::after{content: "★";}

  • ::first-letter

    指定ブロック要素の最初の文字にスタイルを適用

    p::first-letter{color: yellow;}

  • ::selection

    指定要素が選択がされたときにスタイルを適用

    div::selection{color: yellow;}

(6)優先度

Top Before Next Last

同一条件のセレクタについては、後に記述したスタイルが優先される。

p{color:red;}

p{color:blue}

上記の場合color:blueが適用される。

異なる条件のセレクタの場合、下記優先ルールに従い、値の高いスタイルが優先される。 なお、!importantを指定すると最優先となる。

p{color: red!important;}

p{color: blue}

color: redが適用される。

■優先ルール
項目 優先ルール
!important指定 最優先
インラインスタイル 1000の位に加算
id 100の位に加算
クラス、属性セレクタ、擬似クラス 10の位に加算
要素、擬似要素 1の位に加算
■優先ルール例
項目 計算方法 合計
ul ol li.red 10の位に1点、1の位に3点 0.0.1.3
#test(id指定) 100の位に1点 0.1.0.0

(7)ボックスモデル

Top Before Next Last

■width,height

要素の幅(width)、高さ(height)を決める。相対単位、もしくは絶対単位で値を指定する。

親要素width:300px;height:300px;
width:50%;height:50%(相対単位)
width:150px;height:50px;(絶対単位)

■max-width,min-width,max-height,min-height

最大幅(max-width)、最小幅(min-widht)、 最大高さ(max-height)、最小高さ(min-height)を決める。 相対単位、もしくは絶対単位で値を指定する。

親要素width:80%;height:300px;
min-width:10%;max-width:50%; min-height:30%;max-height:50%;

■border

枠線の色、スタイル、幅を指定する。 borderプロパティにまとめて指定することも、 border-color、border-style、border-widthプロパティに 個別に指定することも可能。 また、CSS3では角を丸くするborder-radiusプロパティも追加された。

■padding

ボックスモデルの内側4辺に対して余白を指定する。

border:5px solid;width:200px;
5 + 5 + 200 = 210
border:5px solid;width:200px;padding:10px;
5 + 5 + 200 + 10 + 10 = 230
  • 上下左右、個別に設定可能。 
    padding-top: 10px;
padding-right: 20px;
padding-bottom: 10px;
padding-left: 20px;
  • 省略して以下のように記述することも可能(上、右、下、左の順に指定)。 
    padding: 10px 20px 10px 20px;
  • 左右が同じ場合、省略して以下のように記述することも可能(上、左右、下の順に指定)。 
    padding: 10px 20px 10px;
  • 上下、左右が同じ場合、省略して以下のように記述することも可能(上下、左右の順に指定)。 
    padding: 10px 20px;

■margin

ボックスモデルの外側4辺に対して余白を指定する。 指定方法はpaddingと同様。 なお、互いに向き合う方向にmarginを指定すると、どちらか大きい方が使用される。

padding:5px;margin:5px;

■box-sizing

ボックスモデルの計算方法を変更する。

  • content-box

    初期値。width＋border＋paddingが横幅となる。

  • border-box

    widthが横幅となる(width値に収まるよう調整される)。

width:200px;border:5px;padding:5px;height:80px; box-sizing:content-box;(default)
width:200px;border:5px;padding:5px;height:80px; box-sizing:border-box;

■display

  • display:block

    縦に積み重なるように配置される。

  • display:inline

    1行に配置される。

  • display:inline-block

    1行に配置されつつ、固定の幅や高さが指定できる。

  • display:list-item

    li要素のように、先頭に「・」などのリスト記号がついた状態になる。 リスト記号は要素の領域の外に表示されるので、 親要素のpadding-leftが少ない場合、見えなくなってしまう。

  • display:table

    table要素と同じように振舞う。

  • display:table-caption

    caption要素と同じように振舞う。

  • display:table-cell

    td要素と同じように振舞う。

  • display:table-column

    col要素と同じように振舞う。

  • display:table-column-group

    colgroup要素と同じように振舞う。

  • display:table-header-group

    thead要素と同じように振舞う。

  • display:table-footer-group

    tfoot要素と同じように振舞う。

  • display:table-row

    tr要素と同じように振舞う。

  • display:table-row-group

    tbody要素と同じように振舞う。

(8)テキスト装飾

Top Before Next Last

■color

文字の色を指定する。

color:#ff3333;

■font-size

文字サイズを指定する。

font-size:12px;
font-size:1.5em;

■font-weight

文字の太さを指定する。

font-weight:bold;

■font-style

文字の表示スタイルを指定する。

font-style:italic;

■font-family

文字のフォントの種類を指定する。 フォントはカンマで区切って複数指定することができ、 先頭から順番に環境にインストールされているフォントをさがし、 見つかればそのフォントで表示される。

font-family:'sans-serif';
font-family:'arial black';
font-family:'ＭＳ ゴシック';
font-family:'Meiryo';
font-family:'HGP行書体';

■font-variant

スモールキャピタル(フォントサイズはそのままで小文字を大文字にする)で表示する。

font-variant:small-caps;
font-variant:small-caps;

■text-align

水平方向に対する文字の行揃え、均等割付などを指定する。

text-align:left;
text-align:center;
text-align:right;

■text-decoration

文字の装飾を指定する。 text-decoration-line(線の位置)、text-decoration-color(線の色)、 text-decoration-style(線の種類)をまとめて指定できるが、 現状、線の位置以外は多くのブラウザが対応していない。

■float

回り込み配置を行う際に使用する。値にはleft、right、noneがある。 回り込み配置をした際、親要素の横幅に入りきらない場合は 次段に配置される。 floatでleftかrightが指定された要素はフローティングエレメント(浮動要素)と呼ばれ、 浮動要素を囲っている親要素は浮動要素の高さを認識できず、 背景色やborderは浮動要素を囲ってくれない。

    30%
    70%
    <div style="width:250px;border:1px solid;">
  <div style="background-color:#e0ffff;width:30%;">30%</div>
  <div style="background-color:#90ee90;width:70%;">70%</div>
</div>
  • floatあり leftのみ指定
    30%
    70%
    <div style="width:250px;border:1px solid;">
  <div style="background-color:#e0ffff;width:30%;float:left;">30%</div>
  <div style="background-color:#90ee90;width:70%;">70%</div>
</div>
  • floatあり rightのみ指定
    30%
    70%
    <div style="width:250px;border:1px solid;">
  <div style="background-color:#e0ffff;width:30%;">30%</div>
  <div style="background-color:#90ee90;width:70%;float:right;">70%</div>
</div>
  • floatあり left、right指定
    30%
    70%
    <div style="width:250px;border:1px solid;">
  <div style="background-color:#e0ffff;width:30%;float:left;">30%</div>
  <div style="background-color:#90ee90;width:70%;float:right;">70%</div>
</div>

  • left

    右側にある浮動要素との関係を切り離し、下の段に表示される。

  • both

    左右にある浮動要素との関係を切り離し、下の段に表示される。

(14)変形関連のプロパティ

Top Before Next Last

■transform

要素を変形させる。以下の関数をtransformに指定する。 なお3Dの変形の場合、2D変形にZ軸が加わる。 スペース区切りで複数の関数を指定できる。

  • translateX

    横方向に移動する。単位は相対単位(%, em等)、絶対単位(px等)で指定する。

    transform:translateX(20px)
  • translateY

    縦方向に移動する。単位は相対単位(%, em等)、絶対単位(px等)で指定する。

    transform:translateY(20px)
  • translate

    (横方向,縦方向)と指定し縦横方向に移動する。単位は相対単位(%, em等)、絶対単位(px等)で指定する。

    transform:translateY(20px, 20px)
  • scaleX

    横方向に指定倍率で拡大・縮小する。

    transform:scaleX(1.2)
    transform:scaleX(0.8)
  • scaleY

    縦方向に指定倍率で拡大・縮小する。

    transform:scaleY(1.2)
    transform:scaleY(0.8)
  • scale

    (横報告,縦方向)、もしくは(縦横方向)と指定し、指定倍率で拡大・縮小する。

    transform:scaleY(1.2,1.5)
    transform:scale(1.2)
  • rotate

    回転する。単位はrad(ラジアン角)、もしくはdeg(度)で指定する。

    transform:rotate(45deg)
  • skewX

    横方向にゆがませる。単位はrad(ラジアン角)、もしくはdeg(度)で指定する。

    transform:skewX(20deg)
  • skewY

    縦方向にゆがませる。単位はrad(ラジアン角)、もしくはdeg(度)で指定する。

    transform:skewY(20deg)
  • matrix

    3×3の行列計算により、変形させる。指定値は6個。

(15)アニメーション関連のプロパティ

Top Before Next Last

■transition

イベント等を基にアニメーション効果を出す際に指定する。 時間の指定がされている場合、1つめの時間はアニメーション時間、 2つ目の時間は開始遅延時間となる。

hover時にサイズ変更
div.ani{
  width: 100px;
  height: 50px;
  transition-property: width,height,background-color;
  transition-duration: 1s;
  transition-delay: 1s;
  transition-timing-function: ease-in;
}

div:hover{
  width: 200px;
  height: 100px;
  background-color:#ff3333;
}

<div class="ani" style="border:1px solid #000000;overflow:hidden;white-space:nowrap;">hover時にサイズ変更</div>

上記ではwidthをアニメーションさせているが、left、margin、background-colorなど 数値を持つプロパティも指定可能。 transitionはtransition-duration、transition-property、transition-timing-function、 transition-delayをまとめて指定できるショートカット。時間の指定がされている場合、 1つめの時間はアニメーション時間、2つ目の時間は開始遅延時間となる。

  • transition-property

    アニメーションさせるプロパティ(width、marginなど)を指定する。 省略、もしくはallを指定すると、すべてのプロパティがアニメーションの対象となる。

  • transition-duration

    アニメーション時間を指定する。s(秒)、ms(ミリ秒)で指定する。

  • transition-delay

    開始遅延時間を指定する。s(秒)、ms(ミリ秒)で指定する。

  • transition-timing-function

    アニメーションの動作を指定する。

    • ease

      自然な動作。

    • liner

      等速動作。

    • ease-in

      ゆっくり始まる。

    • ease-out

      ゆっくり終わる。

    • ease-in-out

      ゆっくり始まり、ゆっくり終わる。

■animation

アニメーション回数、繰り返しなど、transitionよりもより細かい指定が可能。 @keyframesで開始から終了までの動きを細かく指定し、amimation-nameでそれを参照する。 animationはanimation-name、animation-duration、animation-timing-function、 animation-delay、animation-iteration-count、animation-directionをまとめて指定できる ショートカット。時間の指定がされている場合、1つめの時間はアニメーション時間、 2つ目の時間は開始遅延時間となる。

  height: 100px;
  background-color: #ff3333;

  animation-name: ani2;
  animation-duration: 3s;
  animation-delay: 1s;
  animation-timing-function: ease-in;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

    top: 0px;

  50%{
    left: 100px;
    top: 50px;
  }
  75%{
    left: 0px;
    top: 50px;
  }
  100%{
    left: 0;
    top: 0;
  }
}

  • animation-name

    実行する@keyframesの名前を指定する。

  • animation-duration

    1サイクルのアニメーションの時間を指定する。s(秒)、ms(ミリ秒)で指定する。

  • animation-delay

    開始遅延時間を指定する。s(秒)、ms(ミリ秒)で指定する。

  • animation-iteration-count

    再生回数を指定する。infiniteを指定すると無限再生となる。

  • animation-timing-function

    アニメーションの動作を指定する。

    • ease

      自然な動作。

    • liner

      等速動作。

    • ease-in

      ゆっくり始まる。

    • ease-out

      ゆっくり終わる。

    • ease-in-out

      ゆっくり始まり、ゆっくり終わる。

  • animation-direction

    往復再生を行うかを指定する。往復再生を行う場合、alternateを指定する。

  • animation-fill-mode

    アニメーションが再生中でない場合の状態を指定する。

    • none

      再生中以外は、元々指定されいてる状態となる。

    • forwards

      @keyframesの0%位置で始まり、100%位置で終了する。

    • backwards

      開始時点の位置で始まり、開始時点の位置で終了する。

    • both

      @keyframesの0%の位置で始まり、@keyframesの100%の位置で終了する。
